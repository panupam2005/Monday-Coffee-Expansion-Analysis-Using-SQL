# Monday-Coffee-Expansion-Analysis-Using-SQL
Conducted an end-to-end SQL analysis to identify top cities for Monday Coffee’s expansion. Evaluated revenue, customer demand, rent efficiency, and market potential using joins, CTEs, and window functions. Recommended Pune, Delhi, and Jaipur based on metrics like 7.7M consumers and low rent-per-customer.

Project Title:
Monday Coffee Expansion Analysis Using SQL

Project Description:

• Goal:
Conducted a full SQL-based business analysis to identify the top 3 Indian cities (Pune, Delhi, Jaipur) for Monday Coffee’s offline store expansion by evaluating revenue, customer behavior, population demand, and operational cost efficiency.

• Process:
Developed an end-to-end SQL workflow using joins, CTEs, aggregations, and window functions to analyze city populations, sales transactions, unique customers, coffee consumption estimates, rent distribution, and product-level performance.

• Key Features & Insights:

Estimated market size using demographic modeling (7.7M potential consumers in Delhi).
Identified highest-performing cities based on metrics like ₹10K+ average sale per customer, 69 unique customers, and rent-per-customer as low as ₹156.
Ranked cities using a multi-factor scoring model covering revenue, demand, rent efficiency, and customer activity.
Generated clear, actionable recommendations for business expansion.

• Results:
Pune, Delhi, and Jaipur were recommended as the top expansion markets due to strong demand, high customer spending, and low operational cost.
